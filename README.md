# TrustLens – Digital Trust Engine

### 🛡️ Digital scams exploit human psychology, not just technical vulnerabilities.

TrustLens is a professional-grade trust analysis engine designed to detect psychological manipulation and structural risks in digital communications. It focuses on *why* users fall for scams, providing explainable and actionable insights.

## 🚀 Killer Features

- **🧠 Psychological Attack Detection**: Identifies tactics like Urgency Pressure, Scarcity Manipulation, Reward Bait, and Forced Action Triggers.
- **🧬 Deep Risk Breakdown**: Granular analysis across **Content**, **Link**, and **Intent** intensity.
- **🛡️ Real-time Trust Scoring**: Dynamic 0-100 scoring based on multi-dimensional behavioral heuristics.
- **🔍 Explainable Reports**: Translates technical risk patterns into human-readable warnings.

## 💡 The Solution

Traditional detection systems look for known malicious signatures. TrustLens analyzes the **Intent** behind the message:
- **Content patterns**: Emotional triggers and pressure tactics.
- **Structural risks**: Malformed URLs, shorteners, and masking techniques.
- **Psychological manipulation**: Behavioral exploitation detection.

## ⚙️ Key Insight

> "Scams succeed because they manufacture trust. TrustLens deconstructs that process in real-time."
